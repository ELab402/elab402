How setup IITD Proxy in PC

Set your default web browser automatic proxy configuration to http://proxy.iitd.ac.in/proxy.pac. If you login to IITD network with your Kerberos login and password, you should be able to browse the web immediately. Ensure you purchase a firewall from the computer center to protect your PC from unauthorized access. For proxy settings in specific browsers, follow the instructions below.

Mozilla Firefox

Edit -> Preferences -> Advanced -> Network -> Settings -> Automatic proxy configuration URL

Internet Explorer

Tools -> Internet Options -> Connections -> LAN settings -> Use automatic configuration script

Google Chrome

Settings -> System -> Open your computer's proxy settings. Configure it as specified for your operating system.
